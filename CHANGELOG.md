# Changelog

Todas as mudanças relevantes deste projeto serão registradas neste arquivo.
O formato segue o princípio do [Keep a Changelog](https://keepachangelog.com/pt-BR/1.1.0/)
e o projeto usa versionamento semântico.

## Não publicado

### Adicionado

- `AstrocartographyMap` com mapa vetorial, filtros, zoom, glifos automáticos,
  tooltips responsivos e eventos.
- Tipos públicos e `buildAstrocartographyLines` para gerar linhas `ASC`, `DSC`,
  `MC` e `IC` a partir de posições equatoriais.
- Demo astrocartográfica e ADR sobre a separação entre efemérides e
  renderização.
- Testes unitários com Vitest.
- Lint, formatação e workflow de integração contínua.
- Templates de issues e pull requests.
- Políticas de contribuição, conduta e segurança.
- Guia de publicação em `docs/PUBLISHING.md` e README expandido para consumo
  via npm em apps Vue.

## [0.1.0] - 2026-07-04

### Adicionado

- Matriz do Destino.
- Mandala de mapa natal.
- Carta astrológica horária.
- Domínio astrológico compartilhado, glifos automáticos e tooltips acessíveis.
